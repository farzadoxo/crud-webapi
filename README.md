# CrudWebapi
A Simple CRUD Webapi with Database

**CRUD : CREATE - READ - UPDATE - DELETE**

# Tools
> Python : 3.11

> Sqlite : 3.0

> C# : 13

>EntityFramework : 8.0.0

# How to use ?
**For python 🐍 :**

Install Packages :

Fastapi - Pydantic - Uvicorn or run : `pip install fastapi[standard]`
1. Run database.py for make database file and table
2. Run main.py :)

**For C#💻 :**

1. Run `Program.cs`
